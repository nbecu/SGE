def setDefaultValues_randomChoice(self, mapping):

Description: Set default values with automatic random choice conversion for lists/tuples.

Parameters:

  • mapping: Any
Examples:

"health": ["good", "bad"],
"hunger": ("good", "bad"),
"speed": 3
})
is equivalent to:
setDefaultValues({
"health": lambda: random.choice(["good", "bad"]),
"hunger": lambda: random.choice(("good", "bad")),
"speed": 3
})
