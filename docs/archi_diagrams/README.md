# SGE Architecture Documentation

This directory contains architectural diagrams and technical documentation for SGE.

## Files

- **architecture_diagrams.md** - Core architecture diagrams (Model-View, entity creation, etc.)

## Diagram Types

- **Model-View Architecture** - Core separation of concerns
- **Entity Creation Flow** - How agents and cells are created
- **Class Relationships** - Dependencies and inheritance
- **Game Flow** - Player interactions and game phases
- **Layout Systems** - Enhanced Grid, Vertical, Horizontal layouts

## Usage

Open any `.md` file in Cursor with Markdown Preview Mermaid Support to view the diagrams.
